# @foodshare/ui

Shared UI component library for the FoodShare platform built with React, Radix UI, and Tailwind CSS.

## Installation

```bash
bun add @foodshare/ui
```

## Usage

```tsx
import { Button } from '@foodshare/ui'

export function MyComponent() {
  return (
    <Button variant="default" size="lg">
      Click me
    </Button>
  )
}
```

## Components

- **Button** - Versatile button component with multiple variants

## Development

```bash
# Build
bun run build

# Watch mode
bun run dev

# Type check
bun run type-check
```
